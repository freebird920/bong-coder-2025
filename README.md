## React 프로젝트 생성

1. vscode 실행
2. [파일] - [폴더열기]로 프로젝트 폴더를 연다.
3. <kbd>ctrl</kbd> + <kbd>`</kbd>를 눌려서 터미널을 연다.
4. 
    ```bash
      npm create vite@latest
    ```
   를 입력하고 실행한다.
5. `Project name:`에 적절한 프로젝트 명을 입력한다.
6. 
    ```bash
    ◇  Select a framework:
    │  ○ Vanilla
    │  ○ Vue
    │  ● React # 이거 선택
    ```
7. 
   ```bash
    ◆  Select a variant:
    │  ○ TypeScript
    │  ● TypeScript + SWC # 이거 선택
   ```
8. [파일]-[폴더열기]로 방금 만든 프로젝트 폴더를 연다.
